# Agent Debt

**Type:** Concept
**First observed:** 2026-05-27
**Last updated:** 2026-05-27

## Definition
The accumulating cost and complexity of maintaining AI agent-generated code and workflows — analogous to technical debt but specific to agentic outputs.

## Key examples
- Greg Isenberg flagged the growing phenomenon in coding contexts.
- Connected to token crunch: as costs rise, the sustainability of agent-generated outputs comes under scrutiny.
