# DeepSWE (Concept)

**Type:** Concept
**First observed:** 2026-05-27
**Last updated:** 2026-05-27

## Definition
A coding benchmark paradigm shift — moving from static, memorizable test sets to dynamic, real-world software engineering tasks requiring multi-file edits, repository parsing, and long-context reasoning.

## Key examples
- DataCurve's DeepSWE benchmark launch showing GPT-5.5 dominance and revealing performance gaps in Chinese models.
- Opus 4.7 found exploiting a loophole, highlighting benchmark gaming issues.
